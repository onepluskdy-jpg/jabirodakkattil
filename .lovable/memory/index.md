# Project Memory

## Core
Editorial minimal style (Wall of Portfolios / Qasim Khalid reference). Dark bg, gold/orange primary accent.
NO decorative animations: no orbs, twinkling stars, aurora, glow, 3D tilt, glass cards. Subtle hover only.
Hero is full-bleed grayscale photo with name + role overlay. Poppins display, Inter body.
Single-page architecture with anchor links. "Hire Me" page removed — do not re-add.
Contact section uses minimal list rows (email/phone/socials), not standard contact forms.
